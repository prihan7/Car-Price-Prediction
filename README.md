# 🚗 Car Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts the selling price of used cars using Machine Learning techniques. It involves data preprocessing, exploratory data analysis (EDA), feature engineering, model building, and performance evaluation to identify the factors that influence car prices.

## 🎯 Objective
- Analyze the factors affecting car prices.
- Perform data cleaning and preprocessing.
- Build a regression model to predict car prices.
- Evaluate model performance using regression metrics.

## 📂 Dataset
- **Dataset:** Car Price Dataset
- **File:** `carprice.csv`

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Statsmodels

## 📊 Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Data Preprocessing
6. Model Building
7. Model Evaluation

## 🤖 Machine Learning Model
- Multiple Linear Regression

## 📈 Evaluation Metrics
- R² Score
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

## 📌 Key Insights
- Vehicle specifications significantly influence selling price.
- Feature engineering improved model performance.
- The regression model provides reliable price predictions for unseen data.
