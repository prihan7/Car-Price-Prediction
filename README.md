# 🚗 Car Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts the selling price of cars using Machine Learning techniques. The objective is to identify the factors that significantly influence car prices and build a regression model capable of making accurate price predictions.

## 🎯 Business Problem
A Chinese automobile company, Geely Auto, plans to enter the US automobile market. The goal of this project is to understand the key factors affecting car prices so the company can make informed decisions about vehicle design, pricing, and business strategy.

## 📂 Dataset
- Dataset: Car Price Dataset
- File: `carprice.csv`

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Statsmodels

## 📊 Project Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Model Building
- Model Evaluation

## 📈 Outcome
The project analyzes the important factors affecting car prices and develops a regression model to predict vehicle prices based on different features.
